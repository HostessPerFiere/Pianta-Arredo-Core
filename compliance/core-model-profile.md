# Core Model Compliance Profile

Version: 0.2-draft

Status: Draft

## Required support

A conforming implementation MUST support:

- Project;
- Building;
- Floor;
- Space;
- Room;
- Wall;
- Opening;
- stable identifiers;
- explicit units;
- coordinate references;
- entity lifecycle;
- structured results and errors.

## Required evidence

- schema-validation results;
- test-suite version;
- implementation version;
- known deviations;
- declared optional Capabilities.
